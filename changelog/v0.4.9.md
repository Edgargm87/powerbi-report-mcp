# Changelog — v0.4.9 (2026-04-08)

## [0.4.9] — 2026-04-08

### Added
- **Smart tool loading** — default mode loads only 10 core tools (~2,900 tokens) instead of all 42+ (~13,000 tokens). **82% reduction** in per-turn token overhead.
- `load_tools` meta-tool — lists available on-demand tools and activates them mid-session. Always active in both modes.
- `MCP_TOOLS=all` env var — loads all tools at startup for power users / dev machines.
- Deferred tool registration — tools not in the default set are stored and can be activated at runtime without server restart.
- **ARCHITECTURE.md** — codebase architecture, folder structure, tool registration flow, PBIR read/write, smart tool loading, data flow, error handling.
- **CONTRIBUTING.md** — dev setup, how to add a tool (5-step guide), testing process, QA expectations, PR guidelines, parked features.
- **docs/pbir-gotchas.md** — consolidated all 14 PBIR schema discoveries (B01–B14) with correct formats, JSON snippets, and discovery methods.
- **docs/visual-types.md** — complete visual type reference with naming gotchas, data buckets, special handling, Series bucket guide.
- **docs/example-prompts.md** — 15 example prompts from beginner to advanced, showing tool call sequences.
- **docs/quickstart.md** — 5-minute guide: clone, configure, connect, create page, preview in PBI Desktop.
- **Sample report** — trimmed `pbi report/` to minimal 1-page report with self-contained `financials` semantic model. Ready to clone and test.

### Changed
- Default active tools: `set_report`, `list_pages`, `list_visuals`, `create_page`, `add_visual`, `get_visual`, `format_visual`, `update_visual_bindings`, `set_report_theme`, `bulk_bind`.
- Visual calculation tools (`list_visual_calculations`, `add_visual_calculation`, `delete_visual_calculation`) fully disabled — not registered in any mode. Parked pending PBI Desktop support.
- Bookmark tools remain unregistered (parked).
- README updated: tool loading modes, default vs on-demand tool reference, revised token overhead tables.
- Version bumped to 0.4.9.
