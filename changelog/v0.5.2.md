## Added

### Bookmarks (4 tools ‚Äî unparked)
- `list_bookmarks` ‚Äî list all bookmarks in the report
- `add_bookmark` ‚Äî create a new bookmark with optional page navigation
- `delete_bookmark` ‚Äî delete a bookmark by ID
- `rename_bookmark` ‚Äî rename an existing bookmark

Bookmark tools existed since v0.4.0 but were not registered. Now imported, registered in ALL_TOOLS, and available as on-demand tools via `load_tools`.

### Page Background (1 tool)
- `set_page_background` ‚Äî set canvas background color, wallpaper color, transparency, or clear all background settings

Supports both the page canvas (`objects.background`) and the area behind the canvas (`objects.wallpaper`). Colors specified as hex strings. Transparency 0-100.

### Guide Knowledge Layer (1 tool)
- `guide` ‚Äî serves domain knowledge on demand by topic

Available topics:
- **`svg-visuals`** ‚Äî how SVG visuals work in Power BI (DAX measures as data URIs), where to store them (model vs extension measures), supported visual types, DAX structure conventions, 4 starter templates (progress bar, status pill, sparkline, bullet chart), binding workflow, and community UDF library references
- **`report-design`** ‚Äî layout standards (spacing, margins, typography), the 3-30-300 rule, page layout patterns (executive dashboard, detail/analysis), color usage guidelines, slicer placement, KPI card patterns, and performance tips

The guide tool provides focused, actionable knowledge to help AI agents orchestrate multi-tool workflows. Unlike SKILL.md files (which are loaded every session and are client-specific), the guide tool is on-demand and works with any MCP client.

## Changed

- `PageDefinition` in `pbir.ts` now includes `objects?: Record<string, unknown>` for page-level objects (background, wallpaper, outspacePane)

## Architecture

### New file: `src/tools/guide.ts`
- `registerGuideTool()` ‚Äî registers the `guide` tool
- Topics stored as template literal strings, built lazily inside the register function
- Topic keys are static for use in tool descriptions
- Adding a new topic requires: adding a const string + adding an entry to the `topics` map

- `src/index.ts` ‚Äî import bookmarks + guide, register both, add 6 entries to ALL_TOOLS, update PBIR instructions
- `src/pbir.ts` ‚Äî added `objects` to `PageDefinition` interface
- `src/tools/report.ts` ‚Äî added `set_page_background` tool
